DISO build:

Requirements
============

- Linux / Unix with bash compatible shell
- all common system progs (grep, cat, etc...)
- all common build requirements (libc, libstdc++, g++, make, etc...)
- mkisofs or any compatible tool
- macroassembler AS 1.42 (latest beta) at least (http://john.ccac.rwth-aachen.de:8000/as/)
- zxspectrum utils (https://sourceforge.net/projects/zxspectrumutils/)


Build DEMFIR
============

- download this repository
- extract included DEMFIR source somewhere outside diso directory and enter to it
- check config.a80 and select features you want
- run 'make' and build is done automatically


Build DISO
==========

- go to diso directory
- erase unncessary files (only thing you need are directories 'demos', 'firmwares', 'games', 'utilities' and 'work')
- run command: 'mkisofs -U -V "DISO" -G [path to]demfirA1123_R.bin -o ../diso.iso .' and you will get diso.iso image in parent directory


Use DISO
========

- burn the iso image to CD, RW or store it to FAT formatted CF, SD, HDD
- insert this medium into DEMFIR
- enjoy
